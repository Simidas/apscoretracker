# 定价报告：AP Score Tracker (apscoretracker.com)

## 1. 结论

**推荐方案：纯前端免费工具 + 可选 Pro 高级功能（$4.99/月）**

**核心理由：**
- AP Score Tracker 是纯前端工具（Next.js + localStorage），无 API 成本、无数据库成本
- 竞品中所有 AP 计算器均为免费，直接收费会流失用户
- 差异化价值在"跨模考进度追踪 + 可视化"，而非单次计算
- 目标用户是高中生，付费意愿有限，低价策略更合适

**最大风险：**
- 高中生付费转化率天然偏低（家长付费决策链长）
- 考试季节性明显（5月考试季 vs 暑期淡季）

| 部署成本 | Vercel Hobby 免费档足够（静态导出） |

## 5. 收费模型选择

| 模型 | 适合度 | 原因 |
|---|---|---|
| **订阅制** | ⚠️ 谨慎 | 低频工具，订阅容易 churn；但行业惯例 |
| **Credits 制** | ❌ 不适合 | 无 API 成本，credits 逻辑不成立 |
| **一次性付费** | ⚠️ 可考虑 | 适合低频工具，但难以持续收入 |
| **广告+联盟** | ✅ 推荐 | 零成本工具站的标准变现路径 |
| **免费+Pro 高级功能** | ✅ **最佳** | 基础计算免费，高级追踪/导出收费 |

**推荐：免费 + Pro 高级功能（Freemium）**
- 基础计算器免费（吸引流量、SEO）
- Pro 解锁跨模考追踪、图表导出、无限历史记录

## 6. 免费层设计

**目标：让用户体验到"追踪"的价值，但限制历史记录数量**

| 项目 | 设计 |
|---|---|
| **每日免费次数** | 无限（计算器本身无成本） |
| **功能限制** | 免费用户只能保存最近 **3 次模考记录** |
| **注册要求** | 不需要注册，浏览器 localStorage 自动保存 |
| **防滥用** | IP 限流（同一 IP 过多请求时加验证码） |
| **免费层成本** | ≈ $0（纯前端，无服务器资源消耗） |

**免费层转化钩子：**
- 当用户尝试保存第 4 次模考记录时，弹出提示：
  > "You've reached the free limit. Upgrade to Pro to unlock unlimited exam history and progress tracking."
- 图表预览显示"Pro only"水印，但允许查看最近 3 次的趋势

## 7. 推荐套餐表

| Plan | 价格 | 用量/功能 | 月成本 | 毛利率 |
|---|---|---|---|---|
| **Free** | $0 | 3 次模考记录保存、基础计算器、最近 1 次趋势预览 | $0 | — |
| **Pro** | **$4.99/月** | 无限模考记录、完整进度曲线、知识点热力图、JSON/CSV 导出、无广告 | $0 | **~99%** |
| **Pro Yearly** | **$39.99/年** | 同上，年付 33% 折扣 | $0 | **~99%** |

**Pro 定价依据：**
- 竞品入门价中位数 $6.99/月 × 0.7 = **$4.89** → 取整 **$4.99**
- 低于 Get AP Ready 的 $6.99，降低高中生决策门槛
- 纯前端工具毛利率极高，低价走量策略可行

**NOT-DO（明确不做）：**
- ❌ 不做"无限"套餐（没有成本压力，不需要）
- ❌ 不做 Lifetime（没有持续成本，但也没有持续价值交付）
- ❌ 不做团队/学校版（MVP 阶段聚焦 C 端）
- ❌ 不做 credits 制（无 API 成本，credits 逻辑不成立）

## 8. 年付 / Lifetime 建议

**年付折扣：**
- Pro Yearly: $39.99/年（等效 $3.33/月，**33% 折扣**）
- 心理定价：$4.99 × 12 = $59.88 → 年付 $39.99，感知节省 $20

**Lifetime：**
- **不推荐做 Lifetime**
- 原因：没有持续 API 成本，但也没有持续交付的新价值；Lifetime 会一次性收割未来收入
- 如果用户强烈要求：$79（≈ 月价 × 16），限量 100 名，加 Fair Use 条款

## 10. 压力测试

**场景 1：1000 MAU，5% 转化率**
- 付费用户：50 人
- 月收入：50 × $4.99 = $249.50
- 月成本：$1（域名摊销）
- 毛利率：99.6% ✅

**场景 2：5000 MAU，3% 转化率**
- 付费用户：150 人
- 月收入：150 × $4.99 = $748.50
- 月成本：$1
- 毛利率：99.9% ✅

**场景 3：10000 MAU，2% 转化率**
- 付费用户：200 人
- 月收入：200 × $4.99 = $998
- 月成本：$20（Vercel Pro）
- 毛利率：98% ✅

**结论：纯前端工具几乎没有成本压力，任何转化率都是正毛利。**

## 12. 风险与下一步动作

| 风险 | 等级 | 缓解措施 |
|---|---|---|
| 高中生付费意愿低 | 🔴 高 | 低价 $4.99；年付 $39.99 降低决策成本；考虑家长定向营销 |
| 考试季节性波动 | 🔴 高 | 5月考试季推促销；淡季靠 SEO 内容维持流量 |
| 竞品跟进追踪功能 | 🟡 中 | 快速迭代，建立品牌认知；考虑社区/UGC壁垒 |
| 纯前端无法防盗版 | 🟡 中 | Pro 功能简单，盗版动力低；未来加后端验证 |
| localStorage 数据丢失 | 🟡 中 | Pro 用户可导出 JSON/CSV 备份；未来考虑可选云同步 |

**下一步动作：**
1. **MVP 阶段**：全免费，验证用户是否真的有"追踪"需求
2. **PMF 验证后**：加 Pro 付费墙（3 次记录限制）
3. **增长阶段**：考虑联盟营销（推荐 AP 备考书/课程，Amazon Affiliate）
4. **远期**：加后端 + 用户系统，实现跨设备同步、社交排行榜
