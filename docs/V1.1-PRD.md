## 0. 结论先行

- **做什么**: 3 个 P0 功能 — 目标分数设定、备考建议生成、打印友好样式
- **为什么做**: MVP 上线后核心闭环已通，但用户缺少"目标感"和"行动指引"，留存会差

## 1. 功能规划

### P0 — 目标分数设定 + 差距显示

**需求描述**: 用户可为每个科目设定目标 AP 分数（1-5），系统根据最新记录显示当前与目标的差距。

**交互设计**:
- 在 Score Input Panel 下方新增 "Target Score" 行
- 下拉选择: 1 / 2 / 3 / 4 / 5（默认 3）
- 保存到 localStorage，按 `subjectId` 隔离
- Metric Cards 区域新增第 4 个卡片: "Gap to Target"（目标分 - 最新分，正数显示 "+X"，负数显示 "-X"，达标显示 "✓ Met"）
- Progress Curve 上叠加一条目标分数虚线（YAxis 对应位置）

**数据模型变更**:
```typescript
// 新增 localStorage key
const TARGET_KEY = "apst_targets"; // Record<string, 1|2|3|4|5>
```

**验收标准**:
- [ ] 切换科目时目标分自动加载
- [ ] 保存新记录后 Gap 卡片实时更新
- [ ] 目标分虚线在图表上可见
- [ ] 清除科目数据时保留目标分（不联动清除）

---

### P0 — 备考建议生成（基于薄弱章节）

**需求描述**: 基于 Average Strength 热力图中最低的 2 个 topic，自动生成备考建议文本。

**规则引擎**:
```
IF averageTopicScores 有数据:
  取最低 2 项
  生成建议: "Your weakest areas are {topic1} ({score1}%) and {topic2} ({score2}%). Focus your next study session on {topic1}."
  
  IF 最新 apScore < 目标分:
    追加: "You're {gap} points away from your target AP {target}. Keep grinding!"
  ELSE IF 最新 apScore >= 目标分:
    追加: "You're on track for your target! Maintain your strength in {highest_topic}."
  
  IF 有 notes:
    不覆盖，建议显示在 notes 下方
```

**UI 位置**: History Panel 上方新增 "Study Tips" Panel，深色背景 + 边框高亮

**验收标准**:
- [ ] 保存记录后自动生成建议
- [ ] 无记录时显示占位文案 "Save a practice test to get personalized study tips."
- [ ] 建议文本随新记录实时更新
- [ ] 支持英文-only（首版不做 i18n）

## 2. 技术方案

### 改动文件清单

| 文件 | 改动类型 | 说明 |
|---|---|---|
| `src/lib/tracker-data.ts` | 新增 | `TARGET_KEY`, `getTargetScore()`, `setTargetScore()` |
| `src/components/sections/TrackerApp.tsx` | 修改 | 新增 target score UI、gap metric、study tips panel、目标虚线 |
| `src/app/globals.css` | 新增 | `@media print` 规则 |

### 数据流

```
用户设定目标分 → localStorage (apst_targets)
                           ↓
保存新记录 → 计算 gap → 更新 Metric Card
                           ↓
                  计算最低 topic → 生成 Study Tips
                           ↓
                  渲染 Progress Curve + 目标虚线
```

## 5. 风险与回滚

| 风险 | 缓解措施 |
|---|---|
| localStorage schema 变更导致旧数据不兼容 | 不改动 `apst_records` 结构，新增独立 `apst_targets` key |
| 打印样式影响正常显示 | `@media print` 隔离，仅在打印时生效 |
| 目标虚线与 AreaChart 冲突 | 使用 Recharts `ReferenceLine` 组件 |

**回滚**: 单文件改动，git revert 即可。
